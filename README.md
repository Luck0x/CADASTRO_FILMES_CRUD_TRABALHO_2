<h2> Site para cadastro e Organização de filmes </h2>
<p>Criamos este site com o objetivo de resolver um problema comum: a falta de organização para quem ama assistir a filmes.
Este site é uma plataforma simples e direta onde o usuário pode se cadastrar, fazer login e começar a montar sua própria lista de filmes favoritos de forma super organizada.</p>

<h2> O que o site faz? </h2>
<ul>
<li>Cadastro e Login: Uma tela inicial para o usuário criar sua conta e entrar no sistema com segurança.</li>
<li>Adicionar Filmes: Um formulário prático para cadastrar os filmes que você já viu ou quer ver. </li>
<li>Detalhes do Filme: O site guarda o Nome, Gênero, Ano de Lançamento, Diretor e até o Link da Foto de Capa (para deixar a lista bem visual).</li>
<li>Painel de Exibição: Uma área limpa e organizada onde todos os filmes cadastrados aparecem juntos.</li>
</ul>

<h2> Tecnologias/lignuagens utilizadas</h2>
<ul>
<li>HTML5: Utilizamos para montar toda a estrutura do site, as caixas de texto dos formulários e os botões.</li>
<li>CSS3: Utilizamos para deixar o visual bonito, moderno, com cores legais e garantir que tudo ficasse alinhado na tela.</li>
<li>JavaScript: É o "cérebro" do projeto. Com ele, fizemos a mágica de validar o login, capturar os dados do formulário e exibir os filmes na tela.</li>
</ul>

<h2> Como os dados ficam salvos? (LocalStorage) </h2>
<p>Como este é um projeto focado no visual e na lógica do navegador (Front-end), decidimos usar o **LocalStorage** para simular um banco de dados. 
Isso significa que tudo o que você cadastrar no site fica guardado na memória do próprio navegador. Você pode fechar a aba do site ou desligar o computador; quando voltar, sua lista de filmes e seu usuário ainda estarão lá! No código JavaScript, utilizamos comandos como `JSON.stringify()` para salvar os dados e `JSON.parse()` para recuperá-los.
</p>
